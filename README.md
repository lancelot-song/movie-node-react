# Website-backstage-nodejs

Nodejs+MongoDB网站后台增删改查的简单轮子




## 运行项目(以管理员执行)

npm install

启动 MongoDB

node app.js

[http://localhost:8083/](http://localhost:8083/)  


## 后台 


**模板引擎**  jade 


**数据库**  MongoDB 


**数据库建模工具** mongoose 


**服务器端**  node+express
